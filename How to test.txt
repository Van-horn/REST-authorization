How to launch:
1) git clone https://github.com/Van-horn/REST-authorization
2) cd .\REST-authorization\  
3) Run docker engine
4)"docker-compose up --build" (if an error occurs execute : "docker volume rm rest-authorization_postgres-data", than (3) )

http - http://localhost/authentication/*
https - https://localhost/authentication/*

*
1) /registration
method POST
Headers not needed
Body 
{
  "login": string,
  "email": string,
  "password": string
}

2) /login
method PATCH
Headers not needed
Body 
{
  "email": string,
  "password": string
}

3) /logout
method PATCH
Headers { Authorization : your access token }
Body 
{
  "userId": number,
}

4) /refresh
method PATCH
Headers not needed
Body 
{
  "userId": number,
}

5) /forgotPassword
method PATCH
Headers not needed
Body 
{
  "email": string,
  "password": string
}
